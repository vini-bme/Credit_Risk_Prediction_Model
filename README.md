# Credit_Risk_Prediction_Model

Credit Risk Prediction System:
- Developed a machine learning model to predict loan defaults using financial and behavioral features
- Generated and processed dataset with variables like income, credit score, loan amount, and employment history
- Engineered key features such as debt-to-income ratio and credit utilization
- Trained a Random Forest classifier to classify borrowers into default / non-default categories
- Evaluated model performance using accuracy, precision, recall, and confusion matrix
- Visualized results and risk factors using feature importance and heatmaps

Technologies:
Python, pandas, numpy, scikit-learn, matplotlib, seaborn

Key Insights:
- Credit score and debt-to-income ratio are the strongest indicators of default risk
- Model effectively identifies high-risk borrowers with strong classification performance
- Feature importance provides interpretability for decision-making in lending systems
- Enables scalable and automated risk assessment for loan applications
